# 58 Productions Portfolio Website

Welcome to the official portfolio website for 58 Productions!

## Features

- **Modern Design**: Clean, professional aesthetic with gradient accents
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Engaging animations and transitions
- **Portfolio Showcase**: Grid layout for displaying your projects
- **About Section**: Highlight your achievements and experience
- **Contact Form**: Easy way for clients to get in touch
- **Social Links**: Connect with your audience on social media

## Getting Started

### Local Development

1. Clone the repository
2. Open `index.html` in your web browser
3. Start editing the files to customize with your content

### Customization

#### Update Your Information

- **Logo/Title**: Edit the logo text in `index.html` navbar
- **Hero Section**: Update the welcome message and CTA button
- **Portfolio**: Edit portfolio items with your projects
- **About Section**: Update your company description and statistics
- **Contact**: The form is ready to integrate with your backend service
- **Social Links**: Add your actual social media URLs in the footer

#### Styling

- Primary colors are defined using gradients in `styles.css`
- Modify colors by editing the gradient values
- Font sizes can be adjusted in the CSS file
